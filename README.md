# MEAN-challenge

## Set up and start

Clone the repo

```
git clone git@github.com:rbecheras/testAngular.git
cd testAngular
```


Install and run the angular front

```
cd front
npm i
npm start
```


Install and run the express server

```
cd back
npm i
npm start
```

## APIs

* `GET /apis/contacts`
* `POST /apis/contacts`
